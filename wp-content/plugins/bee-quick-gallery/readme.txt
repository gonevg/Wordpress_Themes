== Description ==

This Quick Gallery is a responsive image grid gallery based on the gallery by Google for the Chromebook Getting Started guide. In this Quick Gallery we use Masonry for the grid and 3D transforms for navigating the items. For smaller screens we have some example media queries that adjust the grid layout and also the gallery view.

Quick gallery can be use  as portfolio gallery,client gallery,product gallery,photo gallery and more.Gallery is is very easy to use.userfriendly gallery admin section allows you to create a gallery in simple steps.just click on add new gallery and make it and add images to your gallery,also drag and drop for order and save the gallery.copy and pase the gallery shortcode generated automatically.

In quick gallery you can create unlimited gallery with no limits.Also you can use it on unlimited domains.

In the gallery view, the arrow keys can be used to navigate and the view can be closed using the 'Esc' key.

**Features**

*   Fully Responsive gallery
*   Masonry grid gallery
*   3D transforms for navigating
*   The arrow keys can be used to navigate
*   Tile option for eac image
*   Description for each image
*   Wordpress Media supported
*   Drag and sort the images
*   Limit the images per page
*   Enable/Disable option for both title and description

Please note that we are using CSS 3D Transforms which are only supported in modern browsers.

Gallery Plugin installation.


1. Upload `bee-quick-gallery folder` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[beequick id=867]` in your post or pages
